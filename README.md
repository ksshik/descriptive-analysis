Курсовая работа на тему "Дескриптивный анализ численного набора данных с использованием технологий визуализации."
Дисциплина: Машинное обучение

Студент: Дербунович К. А.

Цель: проведение дескриптивного анализа численного набора данных с использованием технологий визуализации и методов машинного обучения для выявления закономерностей, аномалий и взаимосвязей между признаками.

Задачи:

Выбрать и описать числовой датасет.
Выполнить предварительный анализ и очистку данных.
Провести описательный анализ признаков.
Применить методы машинного обучения без учителя.
Обучить и сравнить модели прогнозирования.
Сформулировать выводы и интерпретировать результаты.
